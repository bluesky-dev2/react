# ReactJS Dallas on the web

Home of the ReactJS Dallas user group, founded in February 2016.

## Clone it

`git clone https://github.com/reactjs-dallas/reactjs-dallas-site.git`

## Run it

```sh
cd reactjs-dallas-site
yarn
gatsby develop
```
